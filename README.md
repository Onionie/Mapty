# Mapty

A web application that lets the user log their running and cycling workouts. Built with HTML, CSS, Javacript, Geolocation API and Leaflet.

# Deployment

https://onionie.github.io/Mapty/

# Tools Used

- HTML
- CSS
- Javascript
- Geolocation API
- Leaflet
